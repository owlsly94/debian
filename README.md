# Debian post-install script with XMonad as a WM
XMonad for Debian 12 with install script.
With Catppuccin Mocha colors for XMobar.

## Installation:

```
cd ~
```

```
git clone https://github.com/owlsly94/debian.git
```

```
cd ~/debian/
```

```
chmod +x install.sh
```

```
./install.sh
```

## Enjoy!
